Test for stationary

KPSS Test for Stationarity Description Computes the Kwiatkowski-Phillips-Schmidt-Shin (KPSS) test for the null hypothesis that x is level or trend stationary. See ?kpss.test() additional information

Augmented Dickey-Fuller Test Description Computes the Augmented Dickey-Fuller test for the null that x has a unit root.

See adf.test() additional information
